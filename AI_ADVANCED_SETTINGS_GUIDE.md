## 1. Overview — نظرة عامة {#overview}

### ما هي لوحة AI & Advanced Settings؟

لوحة تحكم متكاملة تسمح لهيئة التدريس/AI بـ:
- تعديل المحتوى التعليمي (الرسائل، الشخصيات، المستويات)
- تعديل أي ملف مصدر في المشروع
- رفع التعديلات مباشرة إلى GitHub
- رفع الملفات إلى Google Drive
- إدارة إعدادات اللعبة المتقدمة

## 3. AI Chat — المحادثة مع AI {#ai-chat}

### ماذا يمكنه AI أن يفعل؟

AI يمكنه تعديل **جميع محتويات اللعبة** تلقائياً + **البحث في الويب** للمعلومات الحديثة:

| المحتوى | مثال على الأمر |
|---------|---------------|
| **الرسائل** | "غيّر رسالة المستوى 3 لتكون عن الفيروسات" |
| **الشخصيات** | "أضف شخصية جديدة اسمها سارة" |
| **المستويات** | "اجعل المستوى 5 أصعب" |
| **الإعدادات** | "غيّر عنوان اللعبة إلى حراس السيبراني" |
| **الشارات** | "أضف شارة جديدة للطلاب المتميزين" |
| **المهام** | "أضف مهمة يومية عن تعلم الشبكات" |
| **الأسئلة** | "أضف سؤالاً جديداً عن حماية كلمات المرور" |
| **الملفات** | "عدّل ملف badges.ts وأضف شارة جديدة" |
| **🔍 بحث في الويب** | "ما هو نموذج GPT-5؟" — يبحث تلقائياً |
| **🧠 تفكير عميق** | "حلل لي مشكلة أمنية معقدة" — يفكر في خطوات |

### خطوات استخدام AI Chat

#### الخطوة 1: افتح AI Chat
```
اضغط على الزر العائم 🤖 → اختر "💬 محادثة AI"
```

#### الخطوة 2: اكتب طلبك
```
مثال: "غيّر رسالة ترحيب المستوى 1 لتكون أكثر حماساً"
```

#### الخطوة 3: AI يعالج الطلب
```
AI يقرأ المحتوى الحالي → يعدّل → يُرسل التعديلات
```

#### الخطوة 4: التعديلات تُطبّق تلقائياً
```
الرسائل ← تُحدث في اللعبة
الشخصيات ← تُحدث في اللعبة
المستويات ← تُحدث في اللعبة
```

### أمثلة على أوامر AI

#### تعديل الرسائل
```
"غيّر رسالة zayn في المستوى 1 لتكون: 
مرحباً! أنا زين، سأعلمك كيف تحمي حاسوبك من الفيروسات"
```

#### إضافة شخصية
```
"أضف شخصية جديدة:
- الاسم: سارة
- الدور: خبيرة الشبكات
- اللون: #E91E63
- الجنس: أنثى
- الشخصية: ذكية ومحبة للتقنية"
```

#### تعديل المستويات
```
"اجعل المستوى 3 أصعب:
- زد النقاط إلى 300
- أضف 5 أسئلة إضافية
- اجعل الوقت أقل"
```

#### إضافة شارة
```
"أضف شارة جديدة:
- الاسم: خبير الشبكات
- الرمز: 🌐
- الشرط: إكمال 3 مهام شبكة
- الوصف: أتقنت أساسيات الشبكات"
```

### كيف يعمل AI داخلياً

```
طلب المستخدم
    ↓
FACULTY_SYSTEM_PROMPT (التعليمات الأساسية)
    ↓
AI يولد JSON بالشكل:
{
  "type": "message",
  "action": "modify",
  "levelId": 1,
  "speakerId": "zayn",
  "field": "outro",
  "lineIndex": 0,
  "data": { "text": "الرسالة الجديدة" }
}
    ↓
applyUpdates() يطبّق التعديلات
    ↓
useContentStore يحفظ التعديلات
    ↓
اللعبة تعرض المحتوى المحدّث
```

### أنواع JSON التي يدعمها AI

#### 1. تعديل الرسائل
```json
{
  "type": "message",
  "action": "modify",
  "levelId": 1,
  "speakerId": "zayn",
  "field": "intro",
  "lineIndex": 0,
  "data": { "text": "الرسالة الجديدة" }
}
```

#### 2. تعديل الشخصية
```json
{
  "type": "character",
  "action": "modify",
  "id": "zayn",
  "data": { "name": "زين الجديد", "role": "خبير الاختراق" }
}
```

#### 3. إضافة شخصية
```json
{
  "type": "character",
  "action": "add",
  "id": "sara",
  "data": {
    "name": "سارة",
    "role": "خبيرة الشبكات",
    "gender": "female",
    "color": "#E91E63",
    "personality": "ذكية ومحبة للتقنية"
  }
}
```

#### 4. حذف شخصية
```json
{
  "type": "character",
  "action": "delete",
  "id": "sara"
}
```

#### 5. تعديل إعدادات اللعبة
```json
{
  "type": "gameMeta",
  "action": "modify",
  "data": {
    "title": "حراس الأمن السيبراني",
    "subtitle": "الإصدار 2.0"
  }
}
```

#### 6. تعديل ملف مصدر (جديد!)
```json
{
  "type": "file",
  "action": "modify",
  "path": "src/data/badges.ts",
  "content": "محتوى الملف الكامل بعد التعديل"
}
```

## 4. Faculty Data Editor — محرر بيانات هيئة التدريس {#editor}

### نظرة عامة

محرر يدوي للمحتوى بدون AI — يسمح بتعديل:
- الإعدادات العامة
- المستويات (7 مستويات)
- الشخصيات (5 شخصيات)
- أي ملف مصدر (جديد!)

### التبويبات

#### 🎮 تبويب "اللعبة" — الإعدادات العامة

| الحقل | الوظيفة | مثال |
|-------|---------|------|
| **العنوان** | عنوان اللعبة | "Cyber Guardians" |
| **العنوان الفرعي** | الوصف | "حراس الأمن السيبراني" |
| **الإصدار** | رقم الإصدار | "2.0.0" |
| **مقياس الصعوبة** | سهولة/متوسط/صعب | "medium" |
| **الصعوبة الافتراضية** | الصعوبة الأولية | "medium" |
| **الوقت لكل سؤال** | بالثواني | "30" |
| **الوقت لكل مستوى** | بالثواني | "600" |
| **مدة اليوم** | بالثواني | "86400" |
| **الحد الأقصى للأرواح** | عدد الأرواح | "5" |
| **التلميحات لكل سؤال** | عدد التلميحات | "3" |
| **موضع HUD** | أعلى/أسفل/يسار/يمين | "top" |
| **نمط القائمة** | شبكة/قائمة/بطاقات | "grid" |
| **سرعة الرسوم** | بطيء/عادي/سريع | "normal" |
| **صوت الموسيقى** | 0-1 | "0.7" |
| **صوت المؤثرات** | 0-1 | "1.0" |
| **صوت الأصوات** | 0-1 | "0.8" |
| **ملاحظات المنصة** | ملاحظات عامة | "ملاحظات..." |

**خطوات التعديل:**
1. اختر التبويب "🎮 اللعبة"
2. عدّل أي حقل
3. اضغط "💾 حفظ"

#### تبويب "المستويات"

**عرض المستويات:**
```
المستوى 1: مقدمة في الأمان — ⭐ 100 نقطة
المستوى 2: كلمات المرور — ⭐ 150 نقطة
...
```

**التعديل:**
1. اختر المستوى من القائمة
2. عدّل العنوان، النقاط، الصعوبة، etc.
3. عدّل الرسائل (intro/outro)
4. اضغط "💾 حفظ"

**مثال على تعديل رسالة:**
```
intro[0].speakerId: "zayn"
intro[0].text: "مرحباً! أنا زين..."
```

#### تبويب "الشخصيات"

**عرض الشخصيات:**
```
زين (zayn) — خبير الاختراق — أزرق
نورة (nora) — خبيرة الشبكات — وردي
...
```

**التعديل:**
1. اختر الشخصية
2. عدّل الاسم، الدور، اللون، الشخصية، الجنس
3. أضف روابط الصور والصوت (اختياري)
4. اضغط "💾 حفظ"

**إعادة ضبط:**
- اضغط "🔄 إعادة ضبط" بجانب أي حقل لإعادته للقيمة الأصلية

## 6. GitHub Integration — ربط GitHub {#github}

### نظرة عامة

ربط اللعبة بمستودع GitHub للحفاظ على التعديلات والنشر.

### الإعداد

#### الخطوة 1: أنشئ مستودعاً على GitHub
```
1. اذهب إلى github.com
2. اضغط "New repository"
3. اختر اسم المستودع (مثلاً: cyber-guardians-mobile)
4. اختر "Public" أو "Private"
5. اضغط "Create repository"
```

#### الخطوة 2: أنشئ Personal Access Token
```
1. اذهب إلى github.com → Settings → Developer settings
2. اضغط "Personal access tokens" → "Tokens (classic)"
3. اضغط "Generate new token"
4. اختر الصلاحيات: repo (كاملة)
5. اضغط "Generate token"
6. انسخ التوكن
```

#### الخطوة 3: أدخل التوكن في اللعبة
```
1. اضغط "⚙ GitHub" في AI Panel
2. أدخل:
   - GitHub Token: ghp_xxxxx
   - Owner: YoussefAhamedKamal
   - Repo: cyber-guardians-mobile
   - Branch: main
3. اضغط "💾 حفظ"
4. اضغط "🔍 اختبار الاتصال"
```

### أزرار GitHub

| الزر | الوظيفة |
|------|---------|
| **🔍 اختبار الاتصال** | يتحقق من صحة التوكن |
| **🔄 رفع إلى GitHub** | يرفع جميع الملفات المعدّلة |
| **📥 استيراد من GitHub** | يجلب أحدث إصدار من GitHub |

### أنواع الرفع

#### 1. رفع المحتوى (JSON)
```
يرفع: gameMeta, characters, dialogue
الصيغة: JSON
الاستخدام: للتعديلات من AI Chat أو Faculty Editor
```

#### 2. رفع الملفات (.ts)
```
يرفع: أي ملف مصدر معدّل
الصيغة: TypeScript
الاستخدام: من تبويب "📁 ملفات"
```

### إنشاء مستودع جديد من اللعبة

إذا لم يكن لديك مستودع:

```
1. اضغط "🔧 إنشاء مستودع جديد"
2. اللعبة تنشئ مستودعاً تلقائياً
3. ترفع جميع الملفات
4. تفعّل GitHub Pages
5. تحصل على رابط النشر
```

## 7. Google Drive Integration — ربط Google Drive {#gdrive}

### نظرة عامة

رفع المحتوى إلى Google Drive كنسخة احتياطية أو للمشاركة.

### الإعداد

#### الخطوة 1: أنشئ Google Cloud Project
```
1. اذهب إلى console.cloud.google.com
2. أنشئ مشروع جديد
3. فعّل Google Drive API
4. أنشئ OAuth 2.0 credentials
5. انسخ Client ID و Client Secret
```

#### الخطوة 2: أدخل البيانات في اللعبة
```
1. اضغط "⚙ Google Drive" في AI Panel
2. أدخل:
   - Client ID: xxxx.apps.googleusercontent.com
   - Client Secret: xxxx
3. اضغط "💾 حفظ"
4. اضغط "🔗 ربط Google Drive"
5. اختر حساب Google
6. اضغط "السماح بالوصول"
```

### أنواع الرفع

#### 1. رفع المحتوى (JSON)
```
يرفع: gameMeta.json, characters.json, dialogue.json
الصيغة: JSON
الاستخدام: للنسخة الاحتياطية
```

#### 2. رفع الملفات (.ts)
```
يرفع: أي ملف مصدر معدّل
الصيغة: TypeScript (.ts)
الاستخدام: للمشاركة مع المطورين
```

### ملاحظات مهمة

- **⚠️** Google Drive API له حدود مجانية
- **⚠️** التوكن ينتهي بعد ساعة — يتجدد تلقائياً
- **⚠️** الملفات تُرفع إلى مجلد "Cyber Guardians" في Drive

## 9. Troubleshooting — حل المشاكل {#troubleshooting}

### مشاكل شائعة

#### ❌ "خطأ 429: Rate limit exceeded"
**السبب:** تجاوزت حد الطلبات المجاني على OpenRouter (50 طلب/يوم)
**الحل:**
```
1. شحن حساب OpenRouter بـ $10 (1000 طلب/يوم)
2. أو استخدم Gemini المجاني (1500 طلب/يوم) — مُوصى به
3. أو انتظر حتى يتجدد العدد (كل 24 ساعة)
```

#### ❌ "خطأ 404: models/gemini-1.5-pro is not found"
**السبب:** نموذج Gemini غير متاح (أُغلق أو غير صحيح)
**الحل:**
```
1. تأكد من اختيار نموذج Gemini 3.x (وليس 2.0 أو 1.5)
2. النماذج المتاحة: gemini-3.5-flash, gemini-3.1-flash-lite, gemini-3-flash
3. النماذج المُغلقة: gemini-2.0-flash, gemini-1.5-pro (أُغلقت يونيو 2026)
```

#### ❌ "البحث لا يُرجع نتائج"
**السبب:** DuckDuckGo لا يُرجع نتائج للاستعلام
**الحل:**
```
1. جرّب استعلاماً مختلفاً
2. تأكد من أن زر 🔍 مفعّل
3. جرّب البحث بكلمات إنجليزية
```

#### ❌ "GitHub token غير مُعد"
**الحل:**
```
1. تأكد من إدخال التوكن
2. تأكد من صحة التوكن (ghp_xxxxx)
3. تأكد من صلاحيات التوكن (repo)
```

#### ❌ "فشل تحميل الملف: 404"
**الحل:**
```
1. تأكد من صحة اسم المستودع
2. تأكد من وجود الملف في المستودع
3. جرّب بدون توكن (raw.githubusercontent.com)
```

#### ❌ "طلب GitHub فشل — تحقق من اتصالك بالإنترنت"
**الحل:**
```
1. تأكد من اتصالك بالإنترنت
2. جرّب إعادة تحميل الصفحة
3. تأكد من عدم حظر CORS
```

#### ❌ "الفيديو متجمد"
**الحل:**
```
1. جرّب متصفح Firefox
2. تأكد من تحميل الفيديو
3. اضغط على الشاشة لتشغيل الفيديو
```

#### ❌ "الترجمات العربية مشفرة"
**الحل:**
```
1. تأكد من دعم UTF-8
2. جرّب متصفح آخر
3. أعد تحميل الصفحة
```

### اختبار الاتصال

1. اضغط "🔍 اختبار الاتصال"
2. إذا ظهر ✅ — الاتصال يعمل
3. إذا ظهر ❌ — تحقق من التوكن

## 🎮 تحديثات اللعبة الأخيرة (v2.1.0)

### إصلاحات مهمة

#### 1. إصلاح نظام المتجر (Shop)
- **المشكلة:** استخدام `game.totalScore` للتحقق من القدرة على الشراء بينما `game.addXp()` يخصم من XP
- **الحل:** استخدام `game.xp` بشكل متسق للتحقق والخصم
- **تحسين:** حفظ المشتريات في localStorage للحفاظ عليها بين الجلسات

#### 2. إصلاح نظام التلميحات (Hints)
- **المشكلة:** زر التلميح كان ينقص العداد فقط بدون عرض محتوى التلميح
- **الحل:** إضافة `onRequestHint` إلى جميع مكونات التحديات
- **تحسين:** عرض نص التلميح الفعلي مع تأثير بصرية

#### 3. تأثيرات بصرية للإجابات
- **الحل:** إضافة وميض أخضر/أحمر عند الإجابة الصحيحة/الخاطئة
- **تحسين:** اهتزاز الشاشة عند الإجابة الخاطئة
- **تحسين:** تأثيرات نبض وتوهج للقلوب والرتب وشريط الخبرة

#### 4. تحسينات واجهة المستخدم
- **HeartsDisplay:** تأثير نبض وتوهج مع خلفية ملونة
- **RankBadge:** تأثير دوران للأيقونة وتوهج للنص
- **XPBar:** تأثير لمعان (shimmer) على شريط التقدم

### كيفية استخدام التلميحات

```
1. أثناء أي تحدي، اضغط على زر "💡 تلميح"
2. يظهر نص التلميح فوق التحدي
3. يختفي تلقائياً بعد 5 ثوانٍ
4. يمكنك استخدام التلميحات في جميع أنواع التحديات:
   - تصنيف الإيميلات (CardChallenge)
   - بناء كلمة المرور (BuildChallenge)
   - متاهة البرمجيات الخبيثة (MazeChallenge)
   - إعداد الجدار الناري (DragDropChallenge)
   - فك التشفير (DecryptChallenge)
   - إصلاح الثغرات (CodeFixChallenge)
   - الاستجابة للحوادث (ResponseChallenge)
```

### كيفية استخدام المتجر

```
1. من القائمة الرئيسية، اضغط على زر "🛒 المتجر"
2. تصفح المنتجات المتاحة:
   - ❤️ تعبئة القلوب (100 نقطة)
   - 💡 حزمة تلميحات (150 نقطة)
   - ⚡ تعزيز XP (200 نقطة)
   - 🌙 سمة ليلية (300 نقطة)
   - ✨ سمة نيون (400 نقطة)
   - 👑 لقب المحترف (500 نقطة)
3. اضغط على "شراء" بجانب المنتج
4. يتم خصم النقاط تلقائياً
5. تُحفظ المشتريات تلقائياً
```
